# VotesApp

## Example
<img src="example.gif" alt="Demo" width="324" height="576">

Let's describe what happens in the video above.
1. User clicks registration button.
2. User inserts username, password, email.
3. User inserts code sent to his email address.
4. User inserts his username.
5. User clicks enter button.
6. User clicks button to offer vote.
7. User offers vote.
8. When administrator accepts user's vote, then user (and other users) can see his vote.
9. User navigates to his vote.
10. User does vote.
11. User gets updated vote according to vote he did.
